**[ES]** **Ovecontroller** es un teclado virtual innovador diseñado específicamente para ser controlado con mandos de juegos (**Xbox, PS4, Joy-Cons**, etc.). Olvídate de la lentitud de escribir carácter a carácter con los sistemas tradicionales de consola; **Ovecontroller** permite una escritura fluida, rápida y cómoda desde la interfaz de tu PC.

**[EN]** **Ovecontroller** is an innovative virtual keyboard designed specifically to be controlled with game controllers (**Xbox, PS4, Joy-Cons**, etc.). Forget the slowness of typing character by character with traditional console systems; **Ovecontroller** allows for fluid, fast, and comfortable typing from your PC interface.

## ✨ Features / Características

- **🎮 Controller Support / Control con Mando**: Compatible with major controllers via optimized input abstraction.
- **⚡ Fast Typing / Escritura Rápida**: Grid design optimized for analog and digital navigation.
- **🔍 Word Suggestions / Sugerencia de Palabras**: Integrated dictionary that learns from your personalized words.
- **🎨 Themes / Temas**: Modern interface with transition effects and interactive carousels.
- **📐 Dynamic Resizing / Redimensionado Dinámico**: Adjust the keyboard size in real-time.
- **🚀 System Integration / Integración con el Sistema**: Launch apps and manage window focus automatically.

## 🛠️ Tech Stack / Stack Tecnológico

- **Tauri**: Rust-based desktop framework.
- **React**: Modern UI library.
- **Typescript**: Robust and maintainable code.
- **Vite**: Ultra-fast build tool.
