**Ovecontroller** es un teclado virtual innovador diseñado específicamente para ser controlado con mandos de juegos (**Xbox, PS4, Joy-Cons**, etc.). Olvídate de la lentitud de escribir carácter a carácter con los sistemas tradicionales de consola; **Ovecontroller** permite una escritura fluida, rápida y cómoda desde la interfaz de tu PC.

## ✨ Características

- **🎮 Control con Mando**: Compatible con los mandos principales mediante una abstracción de entrada optimizada.
- **⚡ Escritura Rápida**: Diseño en cuadrícula optimizado para la navegación analógica y digital.
- **🔍 Sugerencias de palabras**: Diccionario integrado que aprende de tus palabras personalizadas.
- **🎨 Temas**: Interfaz moderna con efectos de transición y carruseles interactivos.
- **📐 Redimensionamiento Dinámico**: Ajusta el tamaño del teclado en tiempo real.
- **🚀 Integración del Sistema**: Lanza aplicaciones y gestiona el foco de las ventanas automáticamente.

---

## 🛠️ Stack Tecnológico

- **Tauri**: Framework de escritorio basado en Rust.
- **React**: Biblioteca de UI moderna.
- **Typescript**: Código robusto y mantenible.
- **Vite**: Herramienta de construcción ultrarrápida.
